libsvm库的使用-SVM分类

ForAgony 4/25/2021

# libsvm库的使用-SVM分类

# 1 作业说明

使用LIBSVM，在西瓜数据集3.0α 上分别用高斯核和拉普拉斯核训练一个SVM，并比较其支持向量的差别。

# 2 LIBSVM库介绍

# 2.1 介绍

LIBSVM是支持向量分类（C-SVC，nu-SVC）、回归（epsilon-SVR，nu-SVR）和分布估计（单类SVM）的集成软件库。

# 2.2 参数介绍

options： 训练的一些参数设置，主要有以下的一些设置

-s SVM的类型(svm_type)

0 -- C-SVC(默认) 使用惩罚因子(Cost)的处理噪声的多分类器

1 -- nu-SVC(多分类器) 按照错误样本比例处理噪声的多分类器

2 -- one-class SVM 一类支持向量机，可参见"SVDD"的相关内容

3 -- epsilon-SVR(回归) epsilon支持向量回归 （也就是使用 E-不敏感损失函数进行回归）

4 -- nu-SVR(回归)

-t 核函数类型(kernel_type)

0 -- linear(线性核): u'*v

1 -- polynomial(多项式核): (gamma*u'*v + coef0)^degree

2 -- radial basis function(RBF,径向基核/高斯核): exp(-gamma*|u-v|^2)

3 -- sigmoid(S型核): tanh(gamma*u'*v + coef0)

4 -- precomputed kernel(预计算核)： 核矩阵存储在training_set_file中

下面是调整SVM或核函数中参数的选项：

-d 调整核函数的degree参数，默认为3

-g 调整核函数的gamma参数，默认为1/num_features

-r 调整核函数的coef0参数，默认为0

-c 调整C-SVC, epsilon-SVR 和 nu-SVR中的Cost参数，默认为1（就是优化问题中的那个C）

-n 调整nu-SVC, one-class SVM 和 nu-SVR中的错误率nu参数，默认为0.5

-p 调整epsilon-SVR的loss function中的epsilon参数，默认0.1

-m 调整内缓冲区大小,以MB为单位，默认100

-e 调整终止判据，默认0.001

-wi 调整C-SVC中第i个特征的Cost参数

调整算法功能的选项：

-b 是否估算正确概率,取值0 - 1，默认为0

-h 是否使用收缩启发式算法(shrinking heuristics),取值0 - 1，默认为0

-v 交叉校验

q 静默模式

# 2.3 输入格式

Libsvm严格定义输入数据的格式。

如果从txt文件中读取数据，数据必须按照以下格式

…

<label> 1:x1 2:x2 …

…

而在python中获取的数据格式为列表嵌套字典。每一个向量组成一个字典，不同向量是一个列表中的元素，即

[{1:x1,2:x2…},…]

# 2.4 自定核函数

从上面参数设置中可以看出，libsvm库中核函数仅有线性核、多项式核、高斯核和sigmoid核，而要求使用的拉普拉斯核函数没有。但是，libsvm库也允许设置参数”-t 4”来使用自定核函数。

在libsvm的说明文档中找到自定核函数部分，如下

Precomputed Kernels

===================

Users may precompute kernel values and input them as training and

testing files. Then libsvm does not need the original

training/testing sets.

Assume there are L training instances x1, ..., xL and.

Let K(x, y) be the kernel

value of two instances x and y. The input formats

are:

New training instance for xi:

<label> 0:i 1:K(xi,x1) ... L:K(xi,xL)

New testing instance for any x:

<label> 0:? 1:K(x,x1) ... L:K(x,xL)

That is, in the training file the first column must be the "ID" of

xi. In testing, ? can be any value.

All kernel values including ZEROs must be explicitly provided. Any

permutation or random subsets of the training/testing files are also

valid (see examples below).

说明，在使用自定核函数时，系统默认输入值为核函数计算结果，而不需要原始数据，因此，自定核函数即为预先计算核函数，再使用计算结果进行svm分类。

需要注意的是，在使用自定核函数时，要在每一项中加上序号，以此判断是哪个向量。

# 3 代码说明

# 3.1 高斯核

from libsvm.svmutil import *


train_label, train_data = svm_read_problem('data.txt')  # 读入训练数据
options = '-t 2'

model = svm_train(train_label, train_data, options)  # 训练
svm_save_model('rbm_model', model)  # 保存训练的模型

如图，从txt文件中读取训练数据，参数设置选择“-t 2”，使用高斯核函数。

默认gamma参数为1/k，即为0.5。

# 3.2 拉普拉斯核

from libsvm.svmutil import *
import numpy as np

导入numpy库来进行矩阵运算。

train_label, train_data = svm_read_problem('data.txt')  # 读入训练数据

data = []
for item in train_data:
    data.append(list(item.values()))
data = np.array(data)
new_data = []

把从文件中读取到的数据进行格式转换，转换到ndarray格式。

for i in range(data.shape[0]):
    item = []
    for j in range(data.shape[0]):
        item.append(kernel(data[i], data[j]))
    new_data.append([i + 1] + item)

分别计算核函数的结果。

def kernel(x, y):
    k = np.linalg.norm(x - y)
    return np.e**(-k/2)

核函数定义，gamma选择和高斯核函数一样的0.5。

nums = [i for i in range(data.shape[0] + 1)]
train_data = []
for i in range(data.shape[0]):
    dic = dict(zip(nums, new_data[i]))
    train_data.append(dic)

将运算结果核矩阵再转换为libsvm要求的输入格式，前面加上序号。

options = '-t 4'    # 参数
model = svm_train(train_label, train_data, options)  # 训练

svm_save_model('laplace_model', model)  # 保存训练的模型

选择参数“-t 4”，使用预先计算核函数。
